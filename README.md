Girdiği bilgi doğru ise ona göre bir sayfaya yönlendiriyor. 
Yetkiye göre güncelleme ya da ekleme işlemi yapılıyor bu projede silme işlemi yok boolen durumda aktiflik pasiflik durumu mevcut.
